Step-by-step explanation

Bubble diameter formula

Three interpretable curves are blended by the emphasis slider: a reference fit, a small-value lift, and a linear-diameter target.

Formula hidden because this range exceeds the recommended use of bounded emphasis.

1. Compare the data range with the marker-size budget

\(M\) describes how far the values spread. \(K\) describes how much visual diameter is available. The reference exponent \(\alpha_0\) is capped at \(0.5\), because \(0.5\) is the area-proportional square-root mapping.

\[ M=\frac{V_{\max}}{V_{\min}},\qquad K=\frac{D_{\max}}{D_{\min}},\qquad \alpha_0=\min\left(0.5,\frac{\ln K}{\ln M}\right) \]

2. Build the reference fit

The reference fit is the neutral curve. If the marker-size budget is tight, \(\alpha_0<0.5\) and the curve compresses the true square-root relation into \(D_{\min}\) through \(D_{\max}\). If the budget is sufficient, \(\alpha_0=0.5\) and the fit equals the raw sqrt reference.

\[ D_{\mathrm{fit}}(v) = D_{\min} \left( \frac{v}{V_{\min}} \right)^{\alpha_0} \]

3. Locate each value in log space

\(x(v)\) runs from \(0\) at \(V_{\min}\) to \(1\) at \(V_{\max}\). This lets the small-value lift preserve both endpoints while changing the curve in between.

\[ x(v)= \frac{\ln(v/V_{\min})}{\ln M} \]

4. Add a bounded small-value lift

When \(e<0\), \(a\) becomes positive and the sine term lifts the middle of the fitted curve without moving \(D_{\min}\) or \(D_{\max}\). The coefficient \(c\) keeps this lift from exceeding the area-proportional exponent \(0.5\).

\[ D_{\mathrm{small}}(v,e) = D_{\mathrm{fit}}(v) \exp\left( \frac{a\,c\,\ln M}{\pi} \sin(\pi x(v)) \right) \] \[ a=\max(-e,0),\qquad c=\max\left(0,\min(\alpha_0,0.5-\alpha_0)\right) \]

5. Define the large-value target

When \(e>0\), the curve moves toward a linear diameter mapping. This gives larger values more diameter separation while still respecting the selected diameter limits.

\[ D_{\mathrm{linear}}(v) = D_{\min} + (D_{\max}-D_{\min}) \frac{v-V_{\min}}{V_{\max}-V_{\min}} \]

6. Blend according to the emphasis slider

Negative \(e\) uses the small-value lift. \(e=0\) keeps the reference fit. Positive \(e\) morphs globally toward the linear-diameter target.

\[ D(v,e) = (1-b)D_{\mathrm{small}}(v,e) + bD_{\mathrm{linear}}(v) \] \[ b=\max(e,0) \]
  • \(e<0\): lifts the fitted curve toward the small-value range.
  • \(e=0\): keeps the reference fit.
  • \(e>0\): morphs the curve toward linear diameter.
  • \(c\): keeps the small-value lift bounded by the area-proportional exponent \(0.5\).
